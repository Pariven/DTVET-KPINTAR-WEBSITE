Expected Flow:

  1. User completes payment ‚Üí Stripe processes payment
  2. Stripe redirects to: /api/payments/success?session_id=cs_...
  3. Success API processes: Updates payment status, clears cart
  4. Success API redirects to: /dashboard/payments?success=true
  5. Dashboard displays: Success message and payment history

Current URLs:

  • Base URL: http://localhost:3000
  • Success URL: /api/payments/success?session_id={CHECKOUT_SESSION_ID}
  • Cancel URL: /checkout?canceled=true
  • Dashboard URL: /dashboard/payments
